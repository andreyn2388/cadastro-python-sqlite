# Cadastro de Usuários com Python e SQLite

Este projeto foi criado como parte dos meus estudos em Python, com o objetivo de praticar lógica de programação e dar meus primeiros passos com banco de dados SQL.

A ideia foi desenvolver algo simples, mas funcional, que me ajudasse a entender como aplicações reais armazenam e consultam dados.

# Tecnologias utilizadas
- Python
- SQLite (biblioteca sqlite3)

# O que o projeto faz
- Recebe dados do usuário (nome e idade)
- Armazena essas informações em um banco de dados SQLite
- Lista os usuários cadastrados

# Aprendizados
Com esse projeto, consegui praticar:
- Entrada de dados com `input()`
- Estruturas básicas do Python
- Integração do Python com banco de dados SQL
- Boas práticas iniciais ao lidar com dados fornecidos pelo usuário
